LinkageSelector (jQuery plugin)

JavaScript 多级联动下拉菜单选择器 Frok me on GitHub

Demo 1

二级联动，子选项为空时默认设置 disabled。

$('#demo1').linkageSelector({
    data: {
        水果: ['西瓜', '柠檬', '草莓', '荔枝', '橘子', '菠萝', '香蕉', '柚子', '苹果', '龙眼'],
        蔬菜: ['香菜', '青菜', '芦笋', '萝卜', '水芹', '黄瓜', '冬瓜', '番茄', '茄子', '土豆']
    }
});

Demo 2

省/市/区三级联动，带默认值，子选项为空时不处理。

$('#demo2').linkageSelector({
    dataUrl: 'https://cdn.rawgit.com/modood/Administrative-divisions-of-China/master/dist/pca.json',
    placeholder: ['省', '市', '区'],
    defaultValue: ['海南省', '三沙市', '西沙群岛'],
    emptyStyle: 'none',
    onChange: function(value) {
        console.log('#demo2', value);
    }
});

$('#demo2 .set').click(function() {
    $('#demo2').linkageSelector('setValue', ['云南省', '昆明市', '呈贡区']);
});

$('#demo2 .reset').click(function() {
    $('#demo2').linkageSelector('resetValue');
});

Demo 3

自动生成 select 元素，无占位项，子选项为空时隐藏，从 code 属性取值。

$('#demo3').linkageSelector({
    dataUrl: 'https://cdn.rawgit.com/modood/Administrative-divisions-of-China/master/dist/pcas-code.json',
    placeholder: false,
    emptyStyle: 'hidden',
    onRender: function(els) {
        [].forEach.call(els, function(el) {
            el.className = 'form-control mr-2';
        });
    },
    onChange: function(value) {
        console.log('#demo3', value);
    }
});
